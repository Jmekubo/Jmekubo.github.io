MSc Project

My MSc project was on predictive analytics. It included a machine learning model developed in Azure machine learning studio, data warehouse hosted in the cloud and a web application to interact with the machine learning model. A tableau dashboard integrated within a web application to present the results.

The solution allowed for prediction of household commodity expenditure for different age categories in the UK.



Main Details:

Data Warehouse developed on SQL Server, Machine learning model utilized linear regression to train model.Other algorithims testes and compared were

Neural Network regression, Boosted Decision Tree Regression, Decision Forest Regression. Linear regression produced the best results and was chosen for the project.

The web application allows for single commodity prediction whereas the console application is used to run batch processing for a list of records(commodities) stored in csv file.

The web application incudes tableau visuals for visualzing results
